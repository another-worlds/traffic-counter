# Counting Line Overlay Module

Status: [IN_PROGRESS]

This module defines the advanced hybrid UI for the Count & Export workflow.

## Responsibilities

- Embed a React/Vite viewport inside Streamlit.
- Synchronize the viewport with selected analyzed videos.
- Let operators create, move, edit, delete, and confirm counting lines.
- Present live counts, overlays, and auto-suggestions.
